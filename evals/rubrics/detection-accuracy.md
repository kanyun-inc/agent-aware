# 检测准确性评估

评估 agent-aware 对预埋问题的检测效果

## 评估标准

### recall_rate (权重: 4)

问题检测的召回率（预埋问题是否都被检测到）

| 分数 | 描述 |
|------|------|
| 4 | 所有预埋问题都被正确检测到 |
| 3 | 检测到 75% 以上的预埋问题 |
| 2 | 检测到 50% 以上的预埋问题 |
| 1 | 检测到少量预埋问题 |
| 0 | 未检测到任何预埋问题 |

### detection_info (权重: 3)

检测信息的准确性（selector、error message 等）

| 分数 | 描述 |
|------|------|
| 3 | 检测信息完整准确，可直接用于定位问题 |
| 2 | 检测信息基本准确，需少量人工确认 |
| 1 | 检测信息部分准确，需要较多人工分析 |
| 0 | 检测信息不准确或缺失 |

### false_positive (权重: 3)

误报率（是否报告了不存在的问题）

| 分数 | 描述 |
|------|------|
| 3 | 无误报，所有检测到的问题都是真实问题 |
| 2 | 少量误报（1-2 个），不影响整体判断 |
| 1 | 较多误报，需要人工筛选 |
| 0 | 大量误报，检测结果不可信 |
