# 代码放置正确性评估

评估 Agent 是否将 SDK 代码放置在正确的位置

## 评估标准

### entry_file_selection (权重: 4)

initAgentAware() 是否放在正确的入口文件中

| 分数 | 描述 |
|------|------|
| 4 | 正确识别并使用了 main.tsx/main.ts/app/layout.tsx 等入口文件 |
| 2 | 放在了可行但非最佳的位置 |
| 0 | 放在了错误的文件中（如组件文件、工具文件） |

### import_position (权重: 3)

import 语句是否放在文件顶部

| 分数 | 描述 |
|------|------|
| 3 | import 语句放在文件顶部，符合规范 |
| 1 | import 位置不标准但能工作 |
| 0 | import 位置错误或遗漏 |

### initialization_timing (权重: 3)

initAgentAware() 是否在应用启动时尽早调用

| 分数 | 描述 |
|------|------|
| 3 | 在应用入口处立即初始化，早于其他代码 |
| 2 | 初始化时机可接受但不是最优 |
| 1 | 初始化时机较晚，可能错过早期行为 |
| 0 | 初始化时机完全错误 |
