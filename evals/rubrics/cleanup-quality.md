# 清理完整性评估

评估 Agent 任务结束时的清理工作

## 评估标准

### sdk_removal (权重: 4)

是否正确移除 SDK 初始化代码

| 分数 | 描述 |
|------|------|
| 4 | 完整移除了 import 语句和 initAgentAware() 调用，没有残留 |
| 2 | 移除了主要代码，有少量残留（如注释） |
| 0 | 没有移除或移除不完整 |

### server_stop (权重: 3)

是否停止了 agent-aware-server 进程

| 分数 | 描述 |
|------|------|
| 3 | 正确执行了停止命令 |
| 1 | 尝试停止但命令不完全正确 |
| 0 | 没有停止服务器 |

### file_cleanup (权重: 3)

是否清理了 .agent-aware 目录（如果用户要求）

| 分数 | 描述 |
|------|------|
| 3 | 按要求清理了临时文件和目录 |
| 2 | 部分清理 |
| 1 | 提示了清理但没有执行 |
| 0 | 没有提及清理 |
